## Getting Started

Inside this `.zip` file, you will find everything needed to launch the project:

### Files included

- `requirements.txt` – Minimal set of packages to run the notebook or scripts efficiently.
- `all_packages.txt` – Full list of all installed packages in the development environment (⚠️ will take longer to install).

---

## Installation Instructions

We recommend starting with the minimal requirements:

```bash
pip install -r requirements.txt

If you run into versioning or missing dependency issues, you can try the full environment:

```bash 
pip install -r all_packages.txt
